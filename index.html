Projek Terbaru

Berikut adalah projek terbaru saya, sebagian besar adalah projek dibidang cloud computing. Tetapi tidak menutup kemungkinan di masa depan akan bertambah pada bidang lain.

Compute Engine

Pada proyek ini saya menggunakan 2 instance vm group yang mana saya deploy di region yang berbeda kemudian saya juga menggunakan custom VPC lalu menghubungkan 2 vm instance menggunakan Cloud NAT. Tak lupa saya juga menggunakan Load Balancer.

App Engine

Saya melakukan deploy untuk aplikasi money tracker menggunakan App Engine di GCP. Money Tracker ini memiliki 2 bagian yaitu frontend dan backend. Masing-masing bagian saya deploy menggunakan App Engine kemudian saya sambungkan pada file codingannya menggunakan alamat yang didapatkan dari masing-masing deployment.
